# cmpNumbers

Number conversions

## BinaryToDecimal
This returns Decimal value of a Binary value.

### Syntax

**BinaryToDecimal**( *Binary* )

- Binary - Required. Type: Number. Binary value. (i.e. 1000001)

### Output

Decimal value (i.e. 65)
